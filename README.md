# htmlcssslide
Simple presentation slides using only HTML and CSS.

Slides are created with plain HTML and are linked to each other using A tags.

Example:

```html
<slide id='slide1'>
    <h1>Your Title</h1>
    <h2>Your Contact Information</h2>
    <nav>
        <a href="">&#8592;</a>
        <a href="#slide2">&#8594;</a>
    </nav>
</slide>
```
